# uas.index
project ini merupakan website kuis online sederhana yang dibuat menggunakan HTML,CSS, dan JavaScript. Website ini menampilkan lima soal pilihan ganda, dimana setiap soal memiliki empat opsi jawaban. Setiap jawaban benar bernilai 20 point dan skor akhir akan dihitung secara otomatis. dan nanti akan menampilkan lulus atau tidaknya.
